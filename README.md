# 图片工具箱

免费在线图片处理工具集，无需上传，即时处理，保护隐私。

## 功能

- **格式转换** — JPG ↔ PNG ↔ WebP
- **图片压缩** — 质量/尺寸压缩
- **裁剪/缩放** — 常用尺寸预设

## 技术栈

- 原生 HTML/CSS/JavaScript
- Canvas API 图片处理
- 纯前端处理，图片不上传服务器

## 本地运行

```bash
# 方式一：Python
python -m http.server 8080

# 方式二：VS Code Live Server 插件
# 右键 index.html → Open with Live Server
```

然后访问 `http://localhost:8080`
